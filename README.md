# NOVA Chatbot

A simple web-based chatbot that interacts with an AI model to generate responses based on user input.

## Features

- Interactive chat interface
- Real-time user input and AI responses
- Message history display
- Simple and clean UI
- API-based communication with an AI model

## Technologies Used

- Python
- HTML
- CSS
- JavaScript
- Anthropic API

## How It Works

- User inputs a message through the interface
- The message is sent to an AI API
- The API processes the input and returns a generated response
- The response is displayed in the chat interface

## Project Purpose

This project was created to explore how AI-powered applications work in practice and to gain hands-on experience integrating AI into a web application.

## Development Approach

This project was developed with the assistance of AI tools to help with implementation and debugging, while I focused on understanding the code, integrating components, and ensuring the application functions correctly.

## Future Improvements

- Improve UI/UX design
- Add conversation memory/context handling
- Support multiple AI models
- Add user authentication
- Deploy as a live web app
